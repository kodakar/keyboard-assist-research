# 修士論文提出要件まとめ（2026年3月修了用）

## 1. 修士論文予稿

### 締め切り
- **提出期間**: 2026年1月13日（火）～ 1月22日（木）17時【厳守】
- 期間内に提出されない場合、予稿集には掲載されない

### フォーマット
- **原稿枚数**: A4版 4枚（片面）
- **ファイル形式**: PDF
- **ファイル名**: `学籍番号.pdf`（例: `24FMI12.pdf`）
- **ページ番号**: 記載しない（予稿集として結合するため）
- **余白**: 左右約2cm
- **カラー**: カラーでも白黒でも可
- **パスワード**: PDFにパスワード設定を一切かけないこと

### 見本フォーマット
```
修士論文テーマ：和文名
修士論文テーマ：英文名
24FMI12 小高大和
研究指導教員 教授 岩井将行

本文開始
```

### 注意事項
- 英文名も忘れずに記載
- 学籍番号・氏名のみ記載（所属専攻や研究室名は記載しない）
- 予稿提出後の修士論文テーマの変更は認められない
- 予稿に記入したテーマが最終テーマとなる

### 提出方法
- 指定のbox URLへアップロード
- 再提出したい場合は東京千住キャンパス事務部大学院担当に連絡

## 予稿テンプレート（yokou.sty）の使い方

### ファイル構成
```
paper/yokou/
├── yokou.sty        ← templete_yokou/からコピー
├── 24FMI12_yokou.tex ← メインファイル
└── (figures は ../../figures/ を参照)
```

### LaTeXの基本構造
```latex
\documentclass[a4paper,twocolumn]{jarticle}
\usepackage{yokou}
\usepackage[dvipdfmx]{graphicx}
\graphicspath{{../../figures/}}

\title{卓上カメラによる指先動作認識を用いた運動機能障害者のためのAIキーボード入力支援システム}
\etitle{AI Keyboard Input Assistance System for People with Motor Impairments Using Fingertip Motion Recognition via Desktop Camera}
\author{24FMI12 小高大和}{研究指導教員 教授 岩井将行}

\begin{document}
\maketitle

% 本文（4ページに収める）

\end{document}
```

### コンパイル方法
```bash
platex -kanji=utf8 24FMI12_yokou.tex
platex -kanji=utf8 24FMI12_yokou.tex
dvipdfmx 24FMI12_yokou.dvi
```

## HCI原稿からの圧縮方針

### HCI原稿（8ページ）の構成
1. はじめに
2. 関連研究
3. 提案システム
4. 評価実験
5. 考察
6. おわりに
7. 参考文献

### 予稿（4ページ）への圧縮案
- 関連研究を大幅に圧縮（詳細な先行研究の説明を削減）
- 考察を「おわりに」に統合
- 図表の数を厳選
- 参考文献を重要なもののみに絞る

### 残すべき重要な内容
- 研究目的と貢献
- システムの全体構成（図1）
- 18次元特徴量の設計
- GRUモデルの選定理由
- 評価実験の結果（表：通常入力83.8%、振戦模擬入力64.9%/90.0%）
- 振戦模擬入力の軌跡可視化（図）
